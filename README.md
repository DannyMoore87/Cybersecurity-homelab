# Self-built Cybersecurity Home Lab

## Overview
A self-built cybersecurity home lab designed to develop hands-on skills in 
system hardening, intrusion detection, access control, and audit logging. 
Built on VirtualBox with multiple virtual machines simulating real-world 
attack and defense scenarios. 

## Lab Environment
- Host OS: Windows with VirtualBox
- Ubuntu 22.04 LTS - Primary target/hardening machine
- Kali Linux - Attack machine
- Metasploitable 2 - Intentionally vulnerable target
- Network: Custom host-only adapter (hacking_lab) + NAT for internet access

## Labs Completed
- User and permission management
- SSH hardening
- UFW firewall configuration
- Apache and MySQL security
- Nmap vulnerability scanning against Metasploitable 2
- Suricata IDS setup and rule configuration
- Lynis security auditing (two rounds, improved hardening score)
- auditd audit logging setup
- AIDE file integrity monitoring

## Skills Demonstrated
- Access control and user permission management
- System hardening and security configuration
- Intrusion detection and monitoring
- Audit logging and recordkeeping
- Vulnerability assessment and scanning
- Network security and firewall management
- File integrity monitoring
## Tools Used
- Lynis - Security auditing and hardening
- Suricata - Intrusion detection system (IDS)
- Nmap - Network scanning and vulnerability assessment
- fail2ban - Brute force protection
- auditd - System audit logging
- AIDE - File integrity monitoring
- UFW - Firewall management
- rkhunter - Rootkit detection
